## Bindings:
* u + ralt = ú
* i + ralt = í
* a + ralt = á
* e + ralt = é
* ó + ralt = ó
* y + ralt = ü
* n + ralt = ñ

## Installation:
* Add "esus" to /usr/share/X11/xkb/symbols
* Add contents of add\_me\_to\_evdev.xml to /usr/share/X11/xkb/evdev.xml (Search for "\<name\>us\</name>" for reference)
